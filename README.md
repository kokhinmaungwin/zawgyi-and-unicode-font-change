# Myanmar Font Converter

This project contains JavaScript functions for converting Myanmar font encodings between Unicode and Zawgyi.

- `Uni_Z1`: Unicode to Zawgyi conversion
- `Z1_Uni`: Zawgyi to Unicode conversion

Usage:

- Open `index.html` in your browser.
- Paste your Myanmar text in the textarea.
- Click the Convert button to switch between Unicode and Zawgyi.
